# アイケアLaBo四ツ谷店 GBP記事制作ルール

## 店舗情報
- 店舗名：アイケアLaBo四ツ谷店
- 所在地：新宿区四ツ谷駅から徒歩4分
- アクセス：市ヶ谷・麹町・曙橋からも好アクセス

## 記事を書く前に必ず参照すること
- references/tone-manner.md（文体・トンマナ）
- references/ng-words.md（禁止表現）
- references/template.md（記事構成テンプレート）

## ターゲット読者
- メイン：30〜50代の社会人・デスクワーカー
- サブ：目の疲れ・老眼・肩こりに悩む方、子どもの視力を心配する親御さん

## 文字数
- 900〜1,100字以内（GBP投稿の最適範囲）

## 必須ルール
- 冒頭は必ず定型文から始める
- 専門用語には必ず読み方を括弧で添える（例：毛様体筋（もうようたいきん））
- 末尾には必ず免責文を挿入する
- 記事はarticles/フォルダにMarkdownファイルで保存する

## 冒頭定型文
こんにちは！新宿区・四ツ谷駅から徒歩4分の「アイケアLaBo四ツ谷店」です👀✨

## 末尾免責文
⚠️ご注意
当店は医療機関ではありません。激しい痛みや異常、長期間続く症状などは、必ず整形外科や眼科等の専門家や医師の診断を受けてください。

#### generate_image.py の変更履歴（詳細）

**① 画像生成プロンプトの改善**（2026-05-07）
のっぺりした画像品質を改善するため `IMAGE_BASE_PROMPT` を刷新。

| 課題 | 追加した指示 |
|---|---|
| 線がのっぺり | 手描き風ペン画の線質・ハッチングを明示 |
| フロー図なし | ①②③矢印付きフロー図を必須要素として追加 |
| アイコンが少ない | 小アイコン（目・星・ハート・稲妻など）6〜10個を散りばめる指示を追加 |
| 強調なし | キーワードの丸囲み・吹き出し強調を追加 |
| 背景が単調 | 薄いグラデーションと生活シーン要素を追加 |
| 色のメリハリ弱い | パステルベース＋強調色1〜2色を明示 |

**② プロンプト生成方式の変更**（2026-05-07）
- Claude CLIによる中間情報抽出（title/scene/pointsのJSON取得）を廃止
- 記事本文をそのままプロンプトに埋め込み gpt-image-2 に直接渡す方式に変更
- `extract_article_info()`・`build_image_prompt(info: dict)` を削除
- 不要な `json` / `re` / `subprocess` インポートを削除
- プロンプトにアクセス情報（四ツ谷駅徒歩4分など）と40〜50代女性向け雰囲気指示を追加

**③ quality・アクセス情報重複・本文トリミングの修正**（2026-05-08）
- `quality="high"` を `images.generate()` に追加
- `IMAGE_BASE_PROMPT` からアクセス情報を削除（記事本文に含まれるため重複解消）
- `trim_article_text()` を追加：冒頭定型文（「こんにちは〜」行）と末尾免責文（`⚠️ご注意` / `---`）を除いた本文のみをプロンプトに渡す
- 実行ログに「本文文字数（定型文・免責文除外後）: XX 字」を表示するよう追加

**④ モデルを gpt-image-2 に変更**（2026-05-09）
- `model="gpt-image-1"` → `model="gpt-image-2"` に変更
- docstring・コメント・ログ内の表記を合わせて更新

#### 現在の generate_image.py の状態

```
実行フロー：
[1/3] 最新記事を読み込み → trim_article_text() で本文のみ抽出
[2/3] gpt-image-2 で画像生成（size=1536x1024, quality=high）
[3/3] images/YYYYMM_タイトル.png に保存
```

#### 技術的な決定事項

- **モデル**：`gpt-image-2`（`gpt-image-1` から変更）
- **画像クオリティ**：`quality="high"`（デフォルト `auto` から変更）
- **画像サイズ**：`1536x1024`（変更なし。`1792x1024` は DALL-E 3 専用）
- **本文トリミング**：冒頭「こんにちは」行と末尾「⚠️」または「---」以降を除去してからプロンプトに渡す
- **アクセス情報**：プロンプトへの直接記載を廃止。記事本文（冒頭定型文）経由で自然に渡る運用に統一
